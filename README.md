# Fullstack Ecommerce Website

A complete ecommerce web application built with **React, JavaScript, HTML ,CSS and used Node.js **


## Features

* Home page had product listing
* Add items to cart from home page
* checkout page had items u have added to cart
* checkout page also had delivery date
* According to date of delivery price is shown on right section of checkout page
* Package tracking page (UI to check order status)
* Unit and Integration testing using **Vitest**
* Splitted each component into its seperate file for clean and reusable


  ## Frontend
  *React
  *HTML5, CSS3, JavaScript
  *Routing with React-Router
  * Vitest for testing
 
  ## Backend
  - Node.js
 
  ## Tools and Deployment
  -Git, GitHub
  -Vercel
